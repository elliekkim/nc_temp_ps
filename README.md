# nc_temp_ps
North Carolina Temperature, Preferentially Sampled

>This repository contains the code used to explore preferential sampling across the State of North Carolina.

## Repository Organization

`\src` contains source code for the models, dataloaders for real and synthetic data.

`\experiments` contains scripts used for experiments.

`\notebooks` contains useful notebooks to showcase model functionality.
